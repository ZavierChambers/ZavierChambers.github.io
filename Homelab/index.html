Overview

My homelab is where I experiment, break, rebuild, and learn. It serves as a training ground for mastering IAM, cybersecurity, server management, and automation tools in real-world environments.

Architecture

Currently running on a mix of physical and virtual machines using Hyper-V and VirtualBox. Core services include a domain controller, file server, Linux VMs, and simulated enterprise environments using open-source tools.

Tech Stack

Active Directory, Windows Server, Ubuntu, Kali Linux, Docker, pfSense, GitHub Actions, and Azure cloud integration. Automation with PowerShell and bash scripts for provisioning, backups, and audits.

Use Cases

Testing GPOs and RBAC, simulating phishing attacks, deploying honeypots, practicing IAM workflows, integrating custom APIs, and simulating attacks and defenses using tools like Splunk and Wireshark.

What's Next

I plan to automate more of my deployments using Infrastructure as Code, integrate SIEM tools, and simulate more cloud/hybrid environments with Entra ID and Microsoft Defender.
